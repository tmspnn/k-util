# k-util
Utility functions of komponenrtize.js

## Installation
```
npm install k-util
```

## Methods
- ```isInt(v: Any): Bool```

- ```toInt(v: Any): Int``` return 0 when v is a falsy value

- ```isString(v: Any): Bool```

- ```toString(v: Any): String``` return '' when v is a falsy value

- ```isObject(v: Any): Bool```

- ```isArray(v: Any): Bool```

- ```toString(v: Any): String``` return '' when v is a falsy value
